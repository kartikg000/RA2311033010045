# RA2311033010045

This is my submission for the Affordmed Campus Hiring Evaluation — Frontend Track.

## What's in here

- `logging_middleware/` — a reusable logging package I built that sends logs to the Affordmed evaluation server
- `notification_app_fe/` — the main React + TypeScript frontend app for the Campus Notifications platform
- `notification_system_design.md` — my approach and design decisions for both stages

## Running the app

```bash
cd notification_app_fe
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.
